    System Architecture
    From raw chest X-ray to binary pneumonia diagnosis
    Model Pipeline
    Input & Preprocessing
    • Chest X-ray (JPEG/PNG)
    • Grayscale → 3-channel RGB
    • CLAHE contrast enhancement
    • Resize → 224 × 224
    • ImageNet normalisation
    • Augmentation (train only)
    ResNet50 Base
    • 50-layer residual network
    • ImageNet pretrained weights
    • Phase 1: base fully frozen
    • Phase 2: top-30 unfrozen
    • Learning rate 1e-4 → 1e-5
    Classification Head
    • GlobalAveragePooling2D
    • BatchNormalization
    • Dense 512 + Dropout 0.4
    • Dense 256 + Dropout 0.3
    • Dense 1 → Sigmoid output
    • Threshold: 0.5
    Training Strategy
    • Binary cross-entropy loss
    • Adam optimiser
    • ReduceLROnPlateau
    • EarlyStopping (val_AUC)
    • ModelCheckpoint (best AUC)
    Grad-CAM XAI
    • conv5_block3_out activations
    • Gradient w.r.t. sigmoid output
    • JET colourmap overlay
    • Highlights infiltrate regions
    Clinical Output
    • NORMAL / PNEUMONIA label
    • Confidence score (%)
    • Risk: Low / Moderate / High
    • Radiological findings list
    • Clinical recommendation
    • Grad-CAM report image
